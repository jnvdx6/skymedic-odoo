[NACEX](https://www.nacex.es) is a leading express courier service in Spain
and Portugal, offering premium, urgent, international, e-commerce, and
pharmaceutical delivery services.

This module integrates NACEX shipping services with Odoo. It allows you to
send shipments, generate shipping labels, and retrieve tracking numbers
directly from delivery orders.

It leverages the NACEX web service API to create expeditions and download
PDF labels automatically upon delivery order validation.
